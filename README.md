# Urban Bites Cafe - Modern Responsive Website

![Urban Bites Cafe Logo](https://img.shields.io/badge/Urban-Bites-5D4037?style=for-the-badge)
![Built for Future Interns](https://img.shields.io/badge/Future%20Interns-Task%203-orange?style=for-the-badge)

A professional, modern, and fully responsive cafe website built for **Urban Bites Cafe**. This project focuses on high-quality aesthetics, smooth user experience, and a clean layout suitable for a real-world local business.

## 🌟 Project Description
“A professional cafe website built for Urban Bites Cafe as part of Future Interns Full Stack Web Development Task 3.”

## ✨ Features
- **Attractive Hero Section**: Eye-catching intro with a professional cafe background.
- **Responsive Navigation**: Smooth transitions and mobile-friendly hamburger menu.
- **Detailed About Us**: Sharing the cafe's story and vision.
- **Interactive Menu**: Categorized food items with high-quality images, descriptions, and pricing.
- **Visual Gallery**: Showcasing the cafe's vibe and delicious dishes.
- **Customer Testimonials**: Social proof from satisfied urban foodies.
- **Contact & Location**: Includes direct contact info and an interactive Google Maps embed.
- **WhatsApp Integration**: Floating "Order Now" button for instant customer interaction.
- **Modern Animations**: Scroll reveal effects and hover transitions for a premium feel.

## 🛠️ Technologies Used
- **HTML5**: Semantic structure for better SEO and accessibility.
- **CSS3 (Vanilla)**: Custom styling with a curated Cream, Brown, and Orange color palette.
- **JavaScript (Vanilla)**: Scroll animations, sticky navbar, and mobile menu logic.
- **Font Awesome**: Premium iconography.
- **Google Fonts**: Modern typography (Playfair Display & Outfit).

## 📦 GitHub Deployment Instructions
1. Push your code to a GitHub repository.
2. Go to **Settings** > **Pages**.
3. Under **Build and deployment**, set the source to `Deploy from a branch`.
4. Select the `main` branch and the `/ (root)` folder.
5. Click **Save**. Your site will be live in a few minutes!
